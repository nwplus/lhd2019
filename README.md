## :construction: Development

```bash

$> cd nwplus2019
$> yarn install
$> yarn start
```

### Web App

The front-end is a React web app.

```bash
$> yarn start        # Serve web app using webpack-dev-server
$> yarn build       # To build the web app into `./docs`
$> yarn serve       # Serve built static web app
```
